# Proxmox Virtualization Homelab

## Overview
This repository documents the design, deployment, and configuration of a multi-node Proxmox virtualization homelab built on Lenovo ThinkCentre systems. The environment is designed to simulate enterprise-grade infrastructure for learning, testing, and professional portfolio development.

The lab integrates:
- Proxmox VE virtualization
- Windows Server Active Directory
- Linux infrastructure services
- Network segmentation and optimization
- Backup and disaster recovery design

This project is intentionally structured as a technical paper and lab documentation series to reflect real-world IT workflows.

---

## Hardware Environment
- Lenovo ThinkCentre Tiny Systems
- Intel-based CPUs
- SSD Storage
- Gigabit Ethernet Networking
- Ubiquiti UCG Ultra Gateway
